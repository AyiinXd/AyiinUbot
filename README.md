### Tutorial Deploy di VPS


 • `git clone https://github.com/AyiinXd/AyiinUbot`

 • `cd AyiinUbot`

 • `pip3 install -U -r requirements.txt`

 • `cp config.env .env`

 • `nano .env`
  - isi vars
  - Jika sudah 
  - ketik ctrl + S
  - ctrl + X

 • `screen -S AyiinUbot`

 • `python3 -m pyAyiin`
